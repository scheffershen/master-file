Définitions

  • PSMF : ou Pharmacovigilance System Master File ou dossier permanent du système de pharmacovigilance
  • Template : Squelette d’un document comportant des balises
  • Document : Généré à partir du Template après remplacement des balises.
  • Balise : Elément représentant une variable, inséré dans un Template, et remplacé automatiquement par sa valeur correspondante à la génération du document.
  • Variable/balise/valeur système : Paramètre dont le scope s’étend sur l’ensemble de tous les PSMF applicables et dont la valeur prise est inhérente à la plateforme (Ex : champ de formulaire / date du jour …).
  • Variable/balise/valeur locale : Paramètre dont le scope d’équivalences est PSMF-dépendant.
  • Variable/balise/valeur globale : Paramètre dont le scope s’étend sur l’ensemble de tous les PSMF applicables et dont la valeur prise est définie pour tous les PSMF.
  • Native : Package élémentaire de contenus et de variables pouvant avoir des comportements et restrictions spécifiques.
  • Table d’équivalences : Lien entre les valeurs et les balises des variables.
